# Markdown Conversion and LLM Interaction

This Streamlit application allows users to convert text files (.txt, .pdf, .docx) to Markdown format and interact with various LLMs (Language Learning Models) for summarization.

## Features

- Upload files or enter text directly.
- Convert uploaded content to Markdown.
- Summarize content using selected models:
  - Without API Key: Facebook BART
  - With API Key: OpenAI GPT-3, Google Gemini, T5, etc.
